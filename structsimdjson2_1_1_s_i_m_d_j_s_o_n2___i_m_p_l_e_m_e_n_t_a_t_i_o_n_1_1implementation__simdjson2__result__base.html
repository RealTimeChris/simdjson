Detailed Description

template<typename T>
struct simdjson2::SIMDJSON2_IMPLEMENTATION::implementation_simdjson2_result_base< T >

The result of a simdjson2 operation that could fail.

Gives the option of reading error codes, or throwing an exception by casting to the desired result.

This is a base class for implementations that want to add functions to the result type for chaining.

Override like:

struct simdjson2_result<T> : public internal::implementation_simdjson2_result_base<T> { simdjson2_result() noexcept : internal::implementation_simdjson2_result_base<T>() {} simdjson2_result(error_code error) noexcept : internal::implementation_simdjson2_result_base<T>(error) {} simdjson2_result(T &&value) noexcept : internal::implementation_simdjson2_result_base<T>(std::forward(value)) {} simdjson2_result(T &&value, error_code error) noexcept : internal::implementation_simdjson2_result_base<T>(value, error) {} // Your extra methods here }

Then any method returning simdjson2_result<T> will be chainable with your methods.
